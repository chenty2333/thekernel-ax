# License texts

The crates in this repository retain their upstream manifest expression:

`GPL-3.0-or-later OR Apache-2.0 OR MulanPSL-2.0`

This directory supplies the corresponding complete texts. They were sourced
from the ArceOS repository at commit
`06f5953e6c2df6a316959972b6b78db78a0db5b6`; exact filenames and checksums are
recorded in [`docs/PROVENANCE.md`](../docs/PROVENANCE.md).

Copies also live in each crate's `LICENSES/` directory so packaged artifacts do
not depend on repository-root files.
